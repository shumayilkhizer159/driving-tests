## Restricted parking

### Signs zone with limited parking time

|  |  |
| --- | --- |
| ![](images/waiting-and-parking-part-3_img_1.jpg) | This sign indicates the beginning of a zone with limited parking time. Such a zone is also called **blue zone**. |

### When

|  |  |
| --- | --- |
| ![](images/waiting-and-parking-part-3_img_2.jpg) | Within this zone, all drivers of a cars must use the **parking disc** on working days when parking their car.  You have to do it **from Monday to Saturday, between 9am and 6pm**, unless otherwise stated on the board.  The parking disc must be clearly displayed behind the front windscreen. |

### How to use a parking disk

|  |  |
| --- | --- |
| ![](images/waiting-and-parking-part-3_img_3.jpg) | On the parking disc you **place the arrow on the dash that follows the time of arrival.** (You arrive at 10h30 so arrow at 11h00.)  You may then **park for two hours**, counting from the time the arrow indicates. Unless a road sign indicates a different time.  After that, you **need to move the car** and not change the time on the parking disc. |

| ![](images/waiting-and-parking-part-3_img_4.jpg) | ![](images/waiting-and-parking-part-3_img_5.png)  Please note: often the parking disc must also be used outside a blue zone. |

## Signs

### Parking allowed

|  |  |

### Sign parking + car

|  |  |
| --- | --- |
| ![](images/waiting-and-parking-part-3_img_9.jpg) | If there is a car under the P, you can park past this sign:   * passenger car; * dual-use car; * minibus; * motorcycle.   (Note the arrow below the board.) |

### Parking on the verge

|  |  |
| --- | --- |
| ![](images/waiting-and-parking-part-3_img_10.jpg) | If you see this sign, it means that if you park, you have to do it **on the verge**.  You can't do it on the carriageway. |

### Parking on the carriageway

|  |  |
| --- | --- |
| ![](images/waiting-and-parking-part-3_img_11.jpg) | If you see this sign, it means that if you park, you have to do it **on the carriageway**.  You can't do it on the verge. |

### Parking partly on the carriageway and partly on the verge

|  |  |
| --- | --- |
| ![](images/waiting-and-parking-part-3_img_12.jpg) | If you see this sign, it means that if you park, you have to do it **partly on the carriageway and partly on the verge**. |

### Alternating parking for certain vehicles

|  |  |
| --- | --- |
| ![](images/waiting-and-parking-part-3_img_13.jpg) | This traffic sign indicates that the parking space is reserved for alternating parking, for example for bicycles and cars.  In this example: from 6:00 p.m. to 7:30 a.m., cars are allowed to park. |

## Getting in or out

### Getting out of the car

|  |  |
| --- | --- |
| ![](images/waiting-and-parking-part-3_img_14.jpg) | Before you get out, you must:   * pull up the handbrake; * close all windows and the open roof; * you turn off the engine; * removes the key from the ignition; * close the doors as soon as you got out. |

### Leaving the car

|  |  |
| --- | --- |
| ![](images/waiting-and-parking-part-3_img_15.jpg) | Leaving the car is done **in the direction of the back of the car**. This way you have a view of the traffic coming from behind. |

### Getting in the car

|  |  |
| --- | --- |
| ![](images/waiting-and-parking-part-3_img_16.jpg) | If you want to get in the car, you **go past the front of the car**. This way you have a view of the traffic coming from behind. |

## Time

### Rules

![](images/waiting-and-parking-part-3_img_17.jpg) ![](images/waiting-and-parking-part-3_img_18.jpg) ![](images/waiting-and-parking-part-3_img_19.jpg)

* A **defective car** may remain on public roads for a **maximum of 24 hours**.
* An **advertising vehicle** may remain on public roads for a **maximum of 3 hours**.
* Offering a **vehicle for sale** on public roads is **prohibited**.
