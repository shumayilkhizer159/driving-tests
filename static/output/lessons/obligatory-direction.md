## Part G: directions

# Lesson 24: Obligatory direction

## Arrow and obstacle

### Slanting arrow signs

![](images/obligatory-direction_img_1.jpg) ![](images/obligatory-direction_img_2.jpg)

On obstacles, command or indication signs with **slanting arrows** can indicate that the obstacle must be **passed in the direction of the arrow**.

This sign can be placed on a pole or on a yellow bollard.

---

## Follow direction

### Command / indication signs

![](images/obligatory-direction_img_3.png) ![](images/obligatory-direction_img_4.png) ![](images/obligatory-direction_img_5.png) ![](images/obligatory-direction_img_6.png) ![](images/obligatory-direction_img_7.png) ![](images/obligatory-direction_img_8.png) ![](images/obligatory-direction_img_9.png)

These **command traffic signs** with arrows indicate that the drivers are **obliged to follow the direction(s) of the arrow(s)**.

### Plate

|  |  |
| --- | --- |
| ![](images/obligatory-direction_img_10.jpg) | A plate can indicate that **cyclists or drivers of mopeds class A** are not obliged to follow the direction indicated by the arrow. |

### Difference

|  |  |
| --- | --- |
| ![](images/obligatory-direction_img_11.jpg) | ![](images/obligatory-direction_img_12.png) ![](images/obligatory-direction_img_13.png)  There is a **big difference** between these two traffic signs.   * At the back you can see an **information sign** which indicates a road with **one-way traffic**. * At the front there is a **command / instruction sign** which indicates the **obligatory direction to follow**. |

|  |  |
| --- | --- |

## Traffic controls

![](images/obligatory-direction_img_15.jpg) ![](images/obligatory-direction_img_16.jpg)

Sometimes there are no signs, but there are **road markings** or **islands** which control the traffic.

You have to pass them **to the right**.
